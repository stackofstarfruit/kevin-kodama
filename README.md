# kevinkodama.com
Personal website built with TypeScript, Tailwind, and SvelteKit. Nice to meet you!

# Tech Stack
- Frontend: TypeScript
- Backend: NodeJS
- UI Framework: Svelte
- CSS Styling: Tailwind
- Build Tool: Vite

# Directory Structure
src/
  components/ # Svelte modular components
  routes/     # Svelte page definitions
  types/      # TypeScript types
static/
  files/      # Non-image static files
  icons/      # SVGs used on the site
  img/        # Static image directory

# Instructions
- Install dependencies with `npm install`
- Local demo available with `npm run dev`
- Generate a new build with `npm run build`
